# e-plantShopping 🌿

## Project Overview
e-plantShopping is a React-based shopping application for house plants.  
It allows users to browse different plants, add them to a shopping cart, and manage quantities before checkout.

## Features
- Browse house plants
- Add plants to shopping cart
- Increase or decrease quantity
- View total price of items
- Dynamic cart updates

## Technologies Used
- React
- Redux Toolkit
- JavaScript
- HTML & CSS
